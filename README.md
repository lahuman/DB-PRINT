## Notice

* JRE 1.7 이상이 설치 되어 있어야 합니다.
* JRE_HOME 환경 변수가 설정 되어 있어야 합니다.
    * 다음이 path에 추가 되어야 함 
		* $JRE_HOME$\bin

## Function
* 특정 테이블만 저장 가능
* 테이블 정보를 HTML or EXCEL로 저장
